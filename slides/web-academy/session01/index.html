04 / 15

하드웨어 vs 소프트웨어

컴퓨터는 몸(하드웨어) 과 정신(소프트웨어)으로 이루어진다.

🔩 하드웨어 (HW)
눈에 보이고 손에 잡히는 물리적 부품
CPU, 메모리(RAM), 저장장치(SSD)
키보드, 마우스, 모니터
전원이 꺼져도 물리적으로 존재
💿 소프트웨어 (SW)
하드웨어에서 실행되는 프로그램과 데이터
운영체제(Windows, macOS)
앱, 브라우저, 문서, 영상
파일 형태로 존재, 삭제 가능
05 / 15
🧠

CPU —
더 깊게 이해하기

CPU는 단순히 "빠른 뇌"가 아니다. 명령어를 받아 처리하는 4단계 사이클을 쉼 없이 반복한다.

1
인출 (Fetch)
메모리에서 다음 실행할 명령어를 가져온다. "다음엔 뭘 해야 하지?"
2
해독 (Decode)
가져온 명령어를 CPU가 이해할 수 있는 형태로 해석한다. "이게 무슨 뜻이지?"
3
실행 (Execute)
계산, 비교, 이동 등 실제 작업을 수행한다. "해냈다!"
4
저장 (Store)
결과를 레지스터나 메모리에 저장한다. 그리고 1단계로 돌아간다.

08 / 15

전원을 누르면
일어나는 일 — 부팅

전원 버튼 하나로 수십 개의 작업이 순차적으로 실행된다.

1
전원 공급
메인보드에 전기가 흐르고, CPU가 깨어난다.
2
BIOS / 펌웨어 실행
하드웨어 상태를 점검한다. RAM, 디스크가 정상인지 확인.
3
운영체제 로딩
SSD에 저장된 Windows/macOS가 메모리(RAM)로 올라온다.
4
로그인 화면
사용자가 앱을 열 수 있는 상태로 준비 완료!
09 / 15
📱

실생활에서
이렇게 쓰여요

우리가 매일 사용하는 앱과 서비스 속에 입력→처리→출력 원리가 녹아 있다.

💬
카카오톡
메시지 입력(키보드) → CPU 처리(인코딩) → 화면 전송(출력)
▶️
유튜브
재생 버튼 클릭(입력) → CPU 처리(디코딩) → 화면에 영상 출력
🔍
네이버 검색
검색어 입력 → 서버 처리(검색 알고리즘) → 결과 페이지 출력

💡 이 개념이 없다면 컴퓨터는 무엇을 해야 할지 전혀 모른다 — 그냥 전기가 흐르는 금속 덩어리일 뿐

10 / 15
💡

자주 하는
오해 바로잡기

이렇게 생각하기 쉽지만, 실제로는 조금 다릅니다.

❌ 흔한 오해
  • 컴퓨터는 뭐든 다 알고 있다
  • 인터넷 = 컴퓨터다
  • 저장 = 기억 (다 똑같다)
✓ 실제로는
  • 명령받은 것만 처리한다. 스스로 생각하지 않는다
  • 인터넷은 별개의 네트워크다. 컴퓨터가 없어도 네트워크는 존재
  • RAM(임시 기억)과 SSD(영구 저장)는 완전히 다르다

👤
CPU 사용률이 100%인데, RAM 부족인지 CPU 과부하인지 어떻게 구분하나요?
🤖
작업 관리자에서 CPU와 메모리 탭을 각각 확인하세요. CPU 그래프가 꽉 차면 처리 과부하, 메모리가 90% 이상이면 RAM 부족입니다. 각각 해결법이 다릅니다!

이렇게
이해하셨나요?

✅
맞게 이해하셨어요
CPU는 컴퓨터의 뇌다 — 모든 계산과 처리를 담당한다
❌
이렇게는 아니에요
RAM이 클수록 무조건 빠른 컴퓨터다 — CPU 속도, SSD 등 여러 요소가 함께 작용한다
✅
맞게 이해하셨어요
입력 → 처리 → 출력은 컴퓨터의 모든 동작을 설명하는 기본 흐름이다
❌
이렇게는 아니에요
소프트웨어는 하드웨어 없이 실행된다 — 반드시 하드웨어 위에서만 동작한다

💪 컴퓨터의 기본 원리를 이해한 것만으로도 엄청난 출발입니다!
