04 / 14

데이터는 어디에 저장될까?

저장 매체는 속도·가격·접근 방식이 모두 다르다.

💽
HDD
자기 디스크가 회전하며 데이터를 읽음. 느리지만 저렴, 대용량에 유리
⚡
SSD
반도체에 직접 저장. 빠르고 조용함, 충격에도 강하지만 가격이 높음
☁️
클라우드
원격 서버에 저장. 어디서나 접근 가능, 인터넷이 필요

절대 경로 vs 상대 경로

같은 파일을 가리키는 두 가지 방식 — 기준점이 다르다.

📍 절대 경로 (Absolute)
루트(/)에서부터 전체 경로
/Users/alice/note.txt
어디서 실행해도 같은 위치
길지만 명확하고 안전
↪️ 상대 경로 (Relative)
현재 위치에서부터의 경로
./note.txt (현재 폴더)
../docs/note.txt (상위 폴더)
짧고 이동에 강함
09 / 14
🧭

경로 기호 —
이렇게 읽어요

처음 보면 낯설지만, 익숙해지면 빠르게 위치를 표현할 수 있다.

/
루트 디렉토리
파일 시스템의 최상위. 모든 경로가 시작되는 지점.
.
현재 디렉토리
지금 내가 있는 폴더. ./file은 "현재 폴더의 file".
..
상위 디렉토리
한 단계 위 폴더. ../로 위로 올라간다.
~
홈 디렉토리
사용자의 개인 폴더. ~/Desktop = 내 바탕화면.

=

자주 하는
오해 바로잡기

이렇게 생각하기 쉽지만, 실제로는 조금 다릅니다.

❌ 흔한 오해
  • 확장자만 바꾸면 파일 형식이 변환된다
  • 클라우드는 데이터를 어딘가 마법처럼 보관한다
  • 파일을 휴지통에서 비우면 즉시 사라진다
✓ 실제로는
  • 확장자는 라벨일 뿐. 변환은 별도 프로그램이 필요하다
  • 클라우드도 결국 누군가의 데이터센터에 있는 SSD/HDD다
  • 물리적으로는 남아있다. 새 데이터로 덮어쓰여야 진짜 사라진다
